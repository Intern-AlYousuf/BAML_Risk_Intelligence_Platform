# Chart Engineering System

All charts must follow these rules:

- Use Recharts only
- Use dark backgrounds
- Use soft gridlines
- Remove clutter
- Use smooth animations
- Use consistent chart heights
- Avoid overcrowding

Forecast charts:

- Show only future forecasts
- Confidence bands should expand gradually
- Use soft yellow fills
- Central forecast line should be clean and visible

Distribution charts:

- Use centered histogram distributions
- Highlight P50 region
- Use muted tails

Axes:

- Minimal tick labels
- No overlapping labels
- Subtle muted text
- Thin gridlines

Legends:

- Compact
- Top-right aligned
- Minimal visual weight

Tooltips:

- Rounded dark containers
- Minimal styling
- High readability

Overall style:
Institutional-grade forecasting dashboard.
