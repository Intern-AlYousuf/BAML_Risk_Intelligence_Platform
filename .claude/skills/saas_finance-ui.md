# SaaS Finance UI System

Design style:
Modern institutional SaaS platform.

References:

- Vercel dashboard
- Stripe dashboard
- Linear
- Ramp
- EY Risk Intelligence example

DO NOT use:

- Bloomberg terminal styling
- Monospace fonts
- Dense tables
- Tiny typography
- Green/red terminal themes
- Grid-heavy layouts
- Overcrowded dashboards

Typography:

- Font family: Inter
- Large titles: bold, clean, minimal
- Compact labels
- Clear hierarchy

Color palette:

- Background: #0B0B0C
- Sidebar: #111318
- Cards: #15171C
- Border: rgba(255,255,255,0.06)
- Primary text: #F5F7FA
- Secondary text: #A1A8B3
- Accent yellow: #F5D90A

Spacing:

- Use consistent 8px spacing scale
- Large whitespace between sections
- Comfortable card padding
- No cramped layouts

Cards:

- Rounded 14px
- Minimal shadows
- Thin borders
- Spacious interiors

Charts:

- Modern Recharts styling
- Minimal axes
- Subtle grids
- Smooth curves
- Soft confidence bands
- Professional institutional feel

Layout:

- Full-height sidebar
- Fixed topbar
- Spacious content container
- Responsive grid system

Overall feel:
Premium modern fintech SaaS platform.
